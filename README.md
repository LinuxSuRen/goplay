# goplay

Listen podcast via CLI.

## Get started

Install via [hd](https://github.com/linuxsuren/http-downloader):

```shell
hd install goplay
```

Start to listen:

```shell
goplay 开源面对面
```

## Release

This project can be released via [linuxsuren-versions](https://github.com/linuxsuren/linuxsuren-versions).
